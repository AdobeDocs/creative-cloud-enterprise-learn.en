# Get started with Firefly Graph

If you've used a generative AI tool before, you're probably used to typing a prompt, getting one result, and starting over from scratch the next time you need something similar. Firefly Graph works differently. Instead of a single prompt, you build a graph: a visual, step-by-step workflow where every input, transformation, and output is connected and saved together. This page walks you through everything you need to build your first graph: the core concepts, a step-by-step first workflow, and a library of ready-made templates you can open and adapt right away.

>[!TIP]
>
>Think of it like a recipe instead of a single meal. Once you've built a graph, you can hand it off, rerun it on new inputs, or change one step and let everything downstream update automatically.
