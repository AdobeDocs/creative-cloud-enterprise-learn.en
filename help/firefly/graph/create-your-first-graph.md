# 3. Create your first Graph

Once you know what a node, connection, and template are, you're ready to build your first workflow.

1. Open Firefly and select **Graph** from the left-hand menu.
1. Select **Create new graph**.
1. Right-click on the blank canvas and select **+ new node**.
1. Select **Input** in the left-hand menu, then **Input Image**.
    ![Node](../assets/node.png)
    This is a node that allows you to import a graphic.
1. Drag and drop an image onto the node.
    ![Node with image](../assets/node-image.png)
1. Right-click on the blank canvas and select **+ new node** and select **Gradient Mask** in the dialog.
1. Right-click on the blank canvas and select **+ new node** and select **Apply Mask** in the dialog.
