# 3. Create your first Graph

Once you know what a node, connection, and template are, you're ready to build your first workflow.

1. **Open a blank canvas, or start from a template.** Browse the node library or the template gallery to find a starting point that's close to what you need.
2. **Connect nodes to build your workflow.** Draw connections from a node's output to the next node's input to chain steps together — for example, an input image flowing into a style node, then into a variation generator, then into export.
3. **Add input controls for anything you'll want to change later.** Turn static values into sliders, color pickers, or text fields so you (or a teammate) can adjust key parameters without editing the underlying graph.
4. **Run the graph and preview each step.** You can preview output at any node before the full workflow finishes running.
5. **Change one input and rerun.** Swap a reference image, adjust a style prompt, or tweak a variable — everything downstream updates automatically. Nothing gets rebuilt by hand.

## What to try first

If you're not sure where to start, open one of the ready-made templates in the template library instead of a blank canvas — it's the fastest way to see a complete, working graph before you build one from scratch.
